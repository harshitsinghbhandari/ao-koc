Agent Orchestrator is a powerful orchestration system for managing parallel AI coding agents. Each agent gets its own git worktree, branch, and PR. The orchestrator itself is an AI agent—not a dashboard, cron job, or polling script. It reads your codebase, understands your backlog, decomposes features into parallelizable tasks, and monitors agent progress autonomously.

See the [original repository](https://github.com/ComposioHQ/agent-orchestrator) for full documentation and architectural details.
